# passwordGenerator

大文字、小文字、数字、アンダーバー（_）が生成されます  
紛らわしい文字と数字などは生成されないようになってます

## 使い方

第1引数はパスワードの桁数。第2引数は何個のパスワードを生成するかというものです。

```bash
$ ./passwordGenerator.sh 16 10
```
